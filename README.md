Distributed RMI Calculator
This project is a simple distributed calculator application implemented using Java Remote Method Invocation (RMI). It allows a client to perform arithmetic operations on a remote server.

Features
Operations: Addition, Subtraction, Multiplication, and Division.
Modern Java Support: Fixed for Java 25 by removing the deprecated SecurityManager.

How to Run
Compile: javac *.java
Registry: Run start rmiregistry in a new window.
Server: Run java CalculatorImpl
Client: Run java CalculatorClient in a separate window.
